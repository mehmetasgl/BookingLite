# BookingLite
 Booking Lite is a hotel reservation web application built with Spring Boot, React.js, PostgreSQL, JWT, and Redis, allowing customers to search hotels and make reservations while partners manage hotels and room availability.

 EN

Booking Lite is a web-based hotel reservation system where customers can search hotels by city and date, view available room types, and create reservations.

The platform also includes partner and admin panels that allow hotel owners to manage hotels, room types, pricing calendars, and reservations.

The project demonstrates a real-world architecture with authentication, role-based authorization, caching, and scalable backend design.

Features

Hotel search by city and date

Room availability and price display

Reservation creation and cancellation

Partner panel for hotel and room management

Admin panel for partner verification and hotel moderation

JWT-based authentication and role-based access control

Redis caching and rate limiting

Technologies

Backend: Java, Spring Boot, Spring Security

Frontend: React.js

Database: PostgreSQL

Authentication: JWT

Cache: Redis

TR

Booking Lite, kullanıcıların şehir ve tarih bilgisine göre otel arayabildiği, uygun oda tiplerini görüntüleyebildiği ve rezervasyon oluşturabildiği web tabanlı bir otel rezervasyon sistemidir.

Bu proje, gerçek dünya uygulamalarına yakın bir mimariyi göstermek amacıyla geliştirilmiştir.

Admin paneli ile partner ve otel moderasyonu
